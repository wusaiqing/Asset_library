本门课程的总成绩主要由平时作业与期末考试构成。

 

    1.上、下学期的课程各有七次作业，每次作业皆算分，每周视频结束后随机做3-8道选择题或简答题，上、下学期分别计分，即不同学期的作业分别计入其所对应的学期，满分各50分。

 

    2.在每个学期的期末各有一次期末考试，各占课程总分的50%，即上学期的期末考成绩占上学期总成绩的50%，下学期的期末考成绩占下学期总成绩的50%。

 

另外，讨论区的互动环节可获得额外加分，具体加分细则将另外发布，烦请各位注意关注。

 

最终成绩≥60者，即可获得合格证书。

 

最终成绩≥85者，即可获得优秀证书。

考核形式：

 

1. 影片小题 (Quizzes)

 

每周的课程影片中，皆会配置若干道简易问题，用以提升修课同学的学习兴趣，此类问题不计入课程总分，请轻松作答，但应注意一点，即必须做答后才可继续观看视频。

 

2. 当周作业 (Homework)

 

配合课程内容，每周皆会有3-8道选择题或简答题，具体的题量将视题型而定。

 

选修学生须在十四天内，也就是课程上线后的两周内，完成并网上提交。其中，客观题即选择题会由系统自动批复，主观题即简答题将在线上进行互评，有关互评的具体要求将另外发布。

 

3. 上学期期末考试 (The First Exam )

 

该考试占上学期课程总成绩的50%；考试期间为上学期最后一次课程上线后两周，只有一次答题机会，提交答案后系统将自动批阅。

 

4. 下学期期末考试 (The Second Exam )

 

该考试占下学期课程总成绩的50%；考试期间为下学期最后一次课程上线后两周，只有一次答题机会，提交答案后系统将自动批阅。
